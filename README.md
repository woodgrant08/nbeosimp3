# NBEO Part 3 Simulator

An interactive web-based exam simulator designed to help optometry students prepare for the NBEO Part 3 (TMOD) clinical examination.

## What it does

- Presents clinical patient cases in a multiple-choice format
- Provides immediate feedback with explanations after each answer
- Covers key optometry topics tested on NBEO Part 3

## How to use it

1. Read the patient case scenario
2. Select your answer from the multiple choice options
3. Get instant feedback and a clinical explanation
4. Move to the next case

## Built with

- HTML
- CSS
- JavaScript

## Project status

This project is actively being developed. Current focus is on building out the question bank and adding features like timed sessions, topic filtering, and score tracking.

## About NBEO Part 3

The National Board of Examiners in Optometry (NBEO) Part 3 — also known as TMOD (Treatment and Management of Ocular Disease) — tests clinical decision-making and patient management skills. This simulator is designed to help students practice in an exam-like environment.
